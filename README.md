# Healthcare Analytics Dashboard

## Project Overview

This project analyzes hospital performance using SQL Server and Power BI.

The dashboard provides insights into:

- Hospital Revenue
- Patient Demographics
- Doctor Performance
- Financial Analytics
- Executive Insights & Recommendations

## Tools Used

- SQL Server
- Power BI
- DAX
- Power Query

## SQL Skills Demonstrated

- Joins
- Aggregations
- Views
- Window Functions

## Dashboard Pages

1. Executive Dashboard
2. Patient Analytics
3. Doctor Performance
4. Financial Analytics
5. Insights & Recommendations

## Key Findings

- Pediatrics generated the highest revenue among all specializations.
- Revenue declined by over 55% from peak month to year-end.
- Sarah was the top-performing doctor in both revenue and appointment volume.
- Adults represented the largest patient demographic.
- Credit Card payments contributed the highest share of collections.

## Recommendations

- Expand Pediatrics services.
- Improve patient retention initiatives.
- Replicate best practices from top-performing doctors.
- Improve collection efficiency.
- Reduce appointment cancellations and no-shows.
